Unidades de Medida en CSS

CSS cuenta con diferentes diferentes unidades para representar longitud.
Muchas propiedades CSS utilizan valores de longitud. Algunos ejemplos de estas propiedades son: width, margin, padding, y font-size
Para utilizar unidades de medida es necesario especificar la cantidad de unidades seguido de la unidad a utilizar.
Por ejemplo:

h1
{
 font-size: 60px;
}

p
{
 font-size: 25px;
 line-height: 50px;
 width: 80%;
}

Existen dos tipos de unidades de medida: relativas y absolutas.
