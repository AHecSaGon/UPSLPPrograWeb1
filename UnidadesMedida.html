Unidades de Medida en CSS

Hector Adrian Sanjuanero Gonzalez - 187283

CSS cuenta con diferentes diferentes unidades para representar longitud.
Muchas propiedades CSS utilizan valores de longitud. Algunos ejemplos de estas propiedades son: width, margin, padding, y font-size
Para utilizar unidades de medida es necesario especificar la cantidad de unidades seguido de la unidad a utilizar.
Por ejemplo:

h1
{
 font-size: 60px;
}

p
{
 font-size: 25px;
 line-height: 50px;
 width: 80%;
}

Existen dos tipos de unidades de medida: relativas y absolutas.


Unidades de medida absolutas

Las unidades de medida absolutas son aquellas cuya longitud siempre es fija. No es recomendado usarlas en situaciones en las que las dimensiones de la pantalla puedan cambiar ya que esto podria romper la visualizacion del contenido.
Las unidades de medida absolutas son las siguientes:

Unidad Descripcion
cm Centimetros
mm Milimetros
in Pulgadas
pt Puntos de impresion (1pt = 1/72 in)
pc Picas (1pc = 12pt)

Algunos ejemplos de su uso son:

p
{
 width: 5cm;
 background-color: aquamarine;
}

Este es un parrafo con un ancho de 5cm

p
{
 width: 45mm;
 background-color: aquamarine;
}

Este es un parrafo con un ancho de 45mm

p
{
 width: 3in;
 background-color: aquamarine;
}

Este es un parrafo con un ancho de 3in


Unidades de medida relativas

Las unidades de medida absolutas son aquellas cuya longitud sera realtiva a algun elemento del sistema. Son recomendadas cuando se sabe que el dispositivo en el que se visualizara la pagina web no siempre sera el mismo
Las unidades de medida relativas son las siguientes:

Unidad Descripcion
em Relativo al tamaño de la fuente
ex Relativo al ancho de la fuente (rara vez utilizado)
ch Relativo al ancho del caracter "0"
rem Relativo al tamaño de la fuente del elemento de raiz (body)
vw Relativo al 1% del ancho de la pestaña
vh Relativo al 1% del alto de la pestaña
vmin Relativo al 1% de la dimension mas pequeña de la pestaña
vmax Relativo al 1% de la dimension mas grande de la pestaña
% Relativo al tamaño del elemento padre

Algunos ejemplos de su uso son:

p
{
 width: 10%;
 background-color: aquamarine;
}

Este es un parrafo con un ancho de 10%

p
{
 width: 10vw;
 background-color: aquamarine;
}

Este es un parrafo con un ancho de 10vw

p
{
 width: 10vh;
 background-color: aquamarine;
}

Este es un parrafo con un ancho de 10vh
